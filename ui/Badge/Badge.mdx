## Usage

```jsx
import { Badge } from "#/ui/components/Badge";
```

```jsx
<Badge>Badge</Badge>
```

## Variants

Each badge has 3 different variants to choose between

<div className="flex flex-wrap gap-2">
  <Badge variant="pill" color="info">
    Pill
  </Badge>
  <Badge variant="rounded" color="info">
    Rounded
  </Badge>
  <Badge variant="square" color="info">
    Square
  </Badge>
</div>

```jsx
<Badge
  variant="pill" // or "rounded" or "square"
  color="info"
>
  Pill
</Badge>
```

## Size

Each badge has 3 different size options to choose between

<div className="flex flex-wrap gap-2">

    sm
  </Badge>
  <Badge size="md" color="info">
    md
  </Badge>
  <Badge size="lg" color="info">
    lg
  </Badge>
</div>

```jsx
<Badge
  size="sm" // or "md" or "lg"

```

## Color

### Random

By default, badges will randomly select a color from the color options

<div className="flex flex-wrap gap-2">
  <Badge>Badge</Badge>
</div>

```jsx
<Badge
  color="random" // or don't pass a color prop
>
  Random
</Badge>
```

### Context

You can also specify a color based on its context

<div className="flex flex-wrap gap-2">
  <Badge color="info">Info</Badge>
  <Badge color="success">Success</Badge>
  <Badge color="warning">Warning</Badge>
  <Badge color="danger">Danger</Badge>
</div>

```jsx
<Badge
  color="info" // or "success" or "warning" or "danger"
>

</Badge>
```

## Transparent

You can pass the transparent prop to remove the badge background color, while leaving the highlight colors

<div className="flex flex-wrap gap-2">
  <Badge color="info" transparent>
    Info
  </Badge>
  <Badge color="success" transparent>
    Success
  </Badge>
  <Badge color="warning" transparent>
    Warning
  </Badge>
  <Badge color="danger" transparent>
    Danger
  </Badge>
</div>

```jsx
<Badge
  color="info" // any color from above
  transparent
>
  Transparent
</Badge>
```

## Icons

You can pass an icon as a child to add an icon to your badge

<div className="flex flex-wrap gap-2">
  <Badge color="info">
    <ArrowLeftIcon /> Info
  </Badge>
  <Badge color="success">
    Info <ArrowRightIcon />
  </Badge>
  <Badge color="warning">
    <AlertTriangleIcon /> Warning
  </Badge>
  <Badge>
    Danger <Announcement01Icon />
  </Badge>
</div>

```jsx
<Badge color="warning">
  <AlertTriangleIcon /> Warning
</Badge>
```

## Interactivity

By default, badges are not interactive but you can pass a prop with interactivity if you need it (don't forget to style it)

<div className="flex flex-wrap gap-2">
  <Badge color="info">
    Remove
    <button className="-mx-1 inline-flex size-5 items-center justify-center rounded hover:bg-black/10 focus:bg-black/10">
      <XCloseIcon className="size-4" />
      <span className="sr-only">Close Example</span>
    </button>
  </Badge>
</div>

```jsx
<Badge color="info">
  Remove
  <button
    onClick={() => alert("Clicked")} // or whatever functionality you need
    className="-mx-1 inline-flex items-center justify-center rounded hover:bg-black/10 focus:bg-black/10 size-5"
  >
    <XCloseIcon className="size-4" />
    <span className="sr-only">Close Example</span>
  </button>
</Badge>
```

## Customization

### Classes

You can customize badges further by passing a className prop, which will override any styles already set.

<div className="flex flex-wrap gap-2">
  <Badge className="rounded-none border-x-0 border-b-2 border-slate-200 border-b-slate-800 bg-slate-200 text-slate-800">
    Destructive
  </Badge>
</div>

```jsx
<Badge className="bg-slate-200 border-x-0 text-slate-800 border-slate-200 border-b-2 rounded-none border-b-slate-800">
  Destructive
</Badge>
```
