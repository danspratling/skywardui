## Usage

```jsx
import { Radio } from "@/ui/Radio";
```

```jsx
<Radio />
```

Note: Radios are note designed to be used alone. You should always use them within a [RadioGroup](/docs/components/RadioGroup).

## Label

You can pass a label to your Radio to automatically associate it with the Radio.

<div className="flex flex-wrap gap-2">
  <Radio label="Accept terms and conditions" defaultChecked />
</div>

```jsx
<Radio label="Accept terms and conditions" defaultChecked />
```

If you need a Radio with accessible text that isn't visible, you can hide the label

<div className="flex flex-wrap gap-2">
  <Radio label="Accept terms and conditions" hideLabel defaultChecked />
</div>

```jsx
<Radio label="Accept terms and conditions" hideLabel defaultChecked />
```

Optionally, if you need your label separated from your Radio, you can associate them by using the `id` and `htmlFor` props.

## Size

You can change the size of your Radio by passing the `size` prop.

<div className="flex flex-wrap gap-2">
  <Radio size="sm" label="size sm" hideLabel defaultChecked />
  <Radio size="md" label="size md" hideLabel defaultChecked />
  <Radio size="lg" label="size lg" hideLabel defaultChecked />
</div>

```jsx
<Radio
  size="sm" // sm, md, lg
  label="size sm"
  hideLabel
  defaultChecked
/>
```

## Icon

You can pass a custom icon if you want to change the default check

<div className="flex flex-wrap gap-2">
  <Radio label="custom icon example" hideLabel defaultChecked>
    <FaceHappyIcon slot="icon" />
  </Radio>
</div>

```jsx
<Radio label="custom icon example" hideLabel defaultChecked>
  <FaceHappyIcon slot="icon" />
</Radio>
```

You may need to adjust the size of your icon to fit the Radio (this isn't done for you for custom icons)

## Controlled

You can control the state of the Radio from outside of the component by passing the `checked` and `onChange` props.

<div className="flex flex-wrap gap-2">
  <Radio label="controlled example" />
</div>

```jsx
export const Controlled = () => {
  const [checked, setChecked] = useState(false);
  return (
    <Radio
      label="controlled example"
      checked={checked}
      onChange={() => setChecked(!checked)}
    />
  );
};
```
